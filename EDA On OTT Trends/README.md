## 📌 Overview

This project performs **Exploratory Data Analysis (EDA)** on OTT platform datasets to uncover trends in digital entertainment. It focuses on analyzing streaming platforms such as **Netflix, Amazon Prime Video, Disney+, Hulu**, and others to understand their content strategies, audience preferences, and growth patterns.

Using Python-based data analysis and visualization libraries, the project transforms raw data into clear and actionable insights.

---

## 🎯 Objectives

The analysis aims to answer questions such as:

- Which OTT platform has the largest content library?
- Are movies or TV shows more common across platforms?
- How has content production changed over the years?
- Which genres dominate each streaming service?
- How do IMDb ratings compare between platforms?
- What trends are emerging in global and regional content?

---

## 📂 Dataset Information

The dataset used for this project contains publicly available OTT content data collected from open sources such as Kaggle.

### Included Features:

- Title  
- Genre  
- Release Year  
- IMDb Rating  
- Content Type (Movie / TV Show)  
- Platform Availability  
- Country of Production  
- Duration  
- Language

---

## 🔍 Key Insights

Some important findings from the analysis:

- 🎬 **Netflix** has one of the largest movie collections.
- 📺 **Prime Video** offers strong TV show variety.
- 📅 Content releases increased rapidly after **2015**.
- 🌍 International and non-English content is growing significantly.
- ⭐ Highly rated content is often found in niche and regional categories.
- 🎭 Drama, Comedy, and Action remain the most common genres.
